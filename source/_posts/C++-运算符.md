C++运算符：

>算术运算符
>关系运算符
>逻辑运算符
>位运算符



## 算术运算符

> a=20 b=10

| 运算符 |               描述               |  实例   |
| :----: | :------------------------------: | :-----: |
|   +    |         把两个操作数相加         | a+b=30  |
|   -    | 从第一个操作数中减去第二个操作数 | a-b=10  |
|   *    |         把两个操作数相乘         | a*b=200 |
|   /    |           分子除以分母           |  a/b=2  |
|   %    |     取模运算符，整除后的余数     |  a%b=0  |
|   ++   |      自增运算符，整数值加 1      | a++=21  |
|   --   |      自减运算符，整数值减 1      | a--=19  |

## 关系运算符

> a=20 b=10

| 运算符 |                             描述                             |    实例     |
| :----: | :----------------------------------------------------------: | :---------: |
|   ==   |       检查两个操作数的值是否相等，如果相等则条件为真。       | a==b(false) |
|   !=   |      检查两个操作数的值是否相等，如果不相等则条件为真。      | a!=b(true)  |
|   >    |   检查左操作数的值是否大于右操作数的值，如果是则条件为真。   |  a>b(true)  |
|   <    |   检查左操作数的值是否小于右操作数的值，如果是则条件为真。   | a<b(false)  |
|   >=   | 检查左操作数的值是否大于或等于右操作数的值，如果是则条件为真。 | a>=b(true)  |
|   <=   | 检查左操作数的值是否小于或等于右操作数的值，如果是则条件为真。 | a<=b(false) |

## 逻辑运算符

> a=1 b=0

| 运算符 |                          描述                          |     实例      |
| :----: | :----------------------------------------------------: | :-----------: |
|   &&   |      与运算符。如果两个操作数都非零，则条件为真。      |  a&&b(false)  |
|  &#124;&#124;  | 或运算符。如果两个操作数中有任意一个非零，则条件为真。 | a&#124;&#124;b(true)  |
|   !    |          非运算符。用来逆转操作数的逻辑状态。          | !(a&&b)(true) |

## 位运算符

> $$
> a=60=(00111100)_2 
> $$
> $$
> b=13=(00001101)_2
> $$


| 运算符 |                          描述                          |     实例      |
| :----- | :------------------------------------- | :---------------- |
| &      | 操作数同位都为0，结果同位为0,反之为1   | a&b=00001100=12   |
| &#124;| 操作数同位都为1，结果同位为1,反之为0   | a&#124;b=00111101=61  |
| ^      | 操作数同位和为1，结果同位为1，反之为0  | a^b=00110001=49   |
| ~      | 操作数0变1，1变0                       | ~a=11000011=-61   |
| <<     | 左操作数的值向左移动右操作数指定的位数 | a<<2=11110000=240 |
| >>     | 左操作数的值向右移动右操作数指定的位数 | a>>2=00001111=15  |
